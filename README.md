# npm-s
> aka **npm quick start**

It creates a `app.js` file and a `package.json` with the following properties in the current directory:
```json
{
  "name": "current-working-directory",
  "version": "0.0.1",
  "scripts": {
    "start": "node app.js"
  }
}
```

## Install
```console
$ npm install -g npm-s
```

## Usage
```console
$ npm-s
```
or
```console
$ npm-start
```
